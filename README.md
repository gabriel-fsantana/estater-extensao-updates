# 📦 Estater - Distribuição da Extensão

Bem-vindo ao repositório de distribuição da extensão **Estater**. 
Este repositório é dedicado a hospedar as versões mais recentes (arquivos `.crx`) e o arquivo de controle de atualizações (`atualizacao.xml`).

Como a nossa extensão é desenvolvida para uso específico e distribuída fora da Google Chrome Web Store, elaboramos uma documentação completa para ajudar os usuários a instalarem a ferramenta de forma rápida e segura.

## 📚 Documentação e Guias

- [📖 Guia de Instalação Passo a Passo](docs/GUIA_DE_INSTALACAO.md) - **Leia isto primeiro para instalar a extensão.**
- [🛠️ Solução de Problemas Comuns](docs/SOLUCAO_DE_PROBLEMAS.md) - Consulte se encontrar erros de bloqueio do Google Chrome.

## 🚀 Versões Disponíveis neste Repositório
- **Versão 1.2** (`extensao_v1.2.crx`) - *Atual*
- **Versão 1.1** (`extensao_v1.1.crx`) - *Anterior*

## 🔄 Como funciona a atualização?
A extensão foi projetada para buscar atualizações consultando o arquivo `atualizacao.xml` hospedado neste repositório. Dependendo do método de instalação escolhido (veja os guias), o navegador poderá fazer o download e atualizar a extensão automaticamente, ou será necessário realizar uma atualização manual baixando a nova versão.
